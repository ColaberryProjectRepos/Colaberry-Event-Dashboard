Adding Events Per Week Visual to the Dashboard 15

Visual 1 : Events Per Week(Line and Stacked Column Chart)

Before adding any visual we need to add a 12 week page level filter to the report as here we are restricting the visuals to display data upto 12 weeks prior to refresh date.Follow the steps below to set a page level filter.

1.Go to filters Pane 

2.On the ‘Filters on this page’ add Slicer Date(from Date Table) 

3.In Filter Type choose ‘Advance Filtering’

4.And on show item when the value is not blank

Follow the gif to Try filter

Now ,We can move forward with adding a  visual 

1.Go to Visualization tab and drag and drop Line and stacked column chart visual to the canvas.

2.Go to Fields tab and add the following

    Shared Axis  : For the Week( Date Table)

                           DayOfWeek (Date Table)

  Column Series : .No of Events per Week(Measures Table) and Rename it to Events Per week  

  Line Values : Rolling 4 Week events(Measures Table) Rename it to Rolling 4 Week

                       Rolling 8 Week events(Measures Table) Rename it to Rolling 8 Week


  • 3.Go to Format tab and apply the following formatting under each tab as mentioned
  •  General:  Turn Responsive on ,X Position = 165, Y Position = 90, Width = 543, Height 311
  •  Legend: On, Position = Top, Title = On, Color = #605E5C, font family = Segoe UI, Text size = 10, Style =  Markers only,Match line color on,circle default icon on
  •  X Axis: Color = #605E5C, Text Size = 9, Font family = Arial , Concatenate labels = On, Title = Off
  •  Y Axis : Position = Left, Scale Type = Linear, Start = Auto, End = Auto, Color = #605E5C, Text size = 9, Font family = Candara, Display Units = Auto, Value Decimal places = Auto, Title = Off,  Gridlines = Off, Show Secondary= on,Align Series =On
  • Data Colors : Rolling 4 week = #12239E,Rolling 8 week : #E66C37,Default color : Click on fx and  in the format style : Field Value,what field should we base this on : events diff 4 and 8
  • Data Labels : Color = #000000,Display units : Auto,Value Decimal places = Auto,orientation= Horizontal,Position =Auto,Text Size = 9,Font Family = segoe UI,Show background = on,Background color =#FFFFFF ,Transparency = 30%,customize series =on, select Rolling 4 weeks-change background color = #D3C5EE ,select Rolling 8 weeks- change background color = #F5C4AF
  • Shapes: Show Marker ON
  • Title : On, Title Text = Events Per Week , Word wrap = On, font color = #FFFFFF, Background color = #000000, Alignment = Center, Text Size = 14, Font Family = Segoe UI Semibold
