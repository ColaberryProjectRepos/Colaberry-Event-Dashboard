Add Logo and Title 13
Add Logo :
  • Must have Colaberry Logo and eventbrite logo  saved  somewhere (zip file from previous step)
  • Click on Insert menu and select Image Then select Colaberry logo from the zip file attached then Resize and adjust the logo .

  • Go to formatting tab > General and make  X - Position = 7, Y - Position = 0, Width = 139 and Height = 91

  • Repeat the same process for the Eventbrite logo and for that also go to Formatting tab> General > and make X - Position = 1169, Y - Position = 0, Width = 111 and Height = 91.

Add Title : 

  • Click on the Text box from Home or insert ribbon.

  • Begin typing the name of the project (EVENTS DASHBOARD)

  • Turn the background to on position: Color : White, Transparency: 0%

  •  Select the text then change the font size to 36  and font type to Candara.

  • General:  X - Position = 362, Y - Position = 6, Width = 556 and Height = 76

Add Refresh Date : 

  • From Visualization select card visual and drop it on the left of the Title text box.

  • Then go to the fields tab and drag the date time column from the Refresh date table .

  • Thereafter go to formatting tab > General and set X - Position = 164, Y - Position = 7, Width = 169  and Height = 76. In the same formatting tab turn category to off. Set font size to 10 and Turn on the Title and right the title name as : Refresh Date and same font size to 10.

    • In the end, Change the page name to ‘Dashboard’
