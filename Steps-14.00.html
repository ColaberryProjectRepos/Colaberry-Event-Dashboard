Adding Cards: Events per Wk

·         Open Visualization Pane

·         Select Visual Card

·         Drag the visual to the dashboard.

·         Click on Events per Wk from Measure Table and drag it into the fields.

·         General: X - Position = 7, Y - Position = 91, Width = 145, Height = 74

·         Data Label: Color = #2F8CA1, Font Size = 20, Font type = Segeo UI

·         Category = OFF

·         Title = ON, Text Title = " Events per Wk ", Font Color = #FFFFF, Background Color = #666666, Font Size = 11 and Font Family = Segeo UI, Alignment = Center

  • Background ON, Color = #FFFFF, Transparency = 0
