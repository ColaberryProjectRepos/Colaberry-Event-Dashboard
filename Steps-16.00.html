Adding Signups Per Week Visual to the Dashboard

Adding a Visual Signups Per Week

1.Go to Visualization tab and drag and drop Line and stacked column chart visual to the canvas.

2.Go to Fields tab and add the following

    Shared Axis  : For the Week( Date Table)

                           DayOfWeek (Date Table)

  Column Series : Count of ‘EvenBrite_EventAttendees[UserId]’ and Rename it to Signups Per Wk 

  Line Values : Signups Rolling 4 Weeks (Measures Table) Rename it to Rolling 4 Week

                      Signups Rolling 8 Weeks (Measures Table) Rename it to Rolling 8 Week


3.Go to Format tab and apply the following formatting under each tab as mentioned

  • General:  Turn Responsive on ,X Position = 721, Y Position =91, Width = 544, Height =310
  • X Axis: Color =#605E5C, Text Size = 9, Font family = Candara,Title = Off

  •  Y Axis :  Color = #605E5C, Text size = 9, Font family = Candara, Title = Off,  Gridlines = Off, Show Secondary= on, Align Series =On,

    Data Colors : Rolling 4 week = #12239E,Rolling 8 week : #E66C37,Default color : Click on fx and  in the format style : Field Value,what field should we base this on : signup diff 4 and 8

    Data Labels : ON Color = #605E5C,Text Size = 9,Font Family = segoe UI, Show background = on, Background color =#FFFFFF ,Transparency = 30%,customize series =on -  Rolling 4 weeks - Show Background = ON, Background Color = #FFFFFF - Rolling 8 weeks -  Show Background = ON, Background Color = #FCDCDC

    Shapes:  Show marker = On,

  • Title = ON Title Text = Signups Per Week,Font color = #FFFFFF, Background color = #000000, Alignment = Center, Text Size = 14pt, Font Family = Segoe UI Semibold
