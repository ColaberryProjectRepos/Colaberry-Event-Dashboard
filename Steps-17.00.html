Adding Signups Per Event Visual to the Dashboard

1.Go to Visualization tab and drag and drop Line and stacked column chart visual to the canvas.

2.Go to Fields tab and add the following

    Shared Axis  : EventBrite_EventGroups(GroupName)

                          EvenBrite_Events(Name)

  Column Series : Measures Table (Total Signups per Week) and Rename it to Signups Per Wk 

  Line Values : Rolling 4 Signups events (Measures Table) Rename it to Rolling 4 Week

                    Rolling 8 Signups events  (Measures Table) Rename it to Rolling 8 Week


3.Go to Format tab and apply the following formatting under each tab as mentioned

  •   General:  Turn Responsive on ,X Position = 165, Y Position = 419, Width = 545, Height =290
  •  X Axis: Color =#605E5C, Text Size = 9, Font family = Candara, Title = Off
  •  Y Axis :  Color = #605E5C, Text size = 9, Font family = Candara, Title = Off,  Gridlines = Off, Show Secondary= on,       Align Series =On
  • Data Colors : Rolling 4 week = #12239E,Rolling 8 week : #E66C37,Default color : Click on fx and  in the format style : Field Value,what field should we base this on : Signups Event diff
  • Data Labels : Color = #605E5C,Display units : Auto,Value Decimal places = Auto,orientation= Horizontal,Position =Auto,Text Size = 9,Font Family = segoe UI,Show background = on,Background color =#FFFFFF ,Transparency = 30%,customize series =on ,Signups Per Wk,show = on,color = #000000 Display units = Auto,Value Decimal places = Auto

  • Shapes:  Show marker = On


    Title = ON Title Text = Signups Per Event ,Font color = #FFFFFF, Background color = #000000, Alignment = Center, Text Size = 14pt, Font Family = Segoe UI Semibold

Finally, click on the "..." on the visual. Sort axis=Signups per week , sort descending
