Adding Refresh Date Table 11

  • Click on Transform data tab in the home ribbon

  • Then in the transform data window that opens up, Go to Home tab > New Source and select Blank Query. Rename the blank query as ‘Refresh Date’.

  • In the formula bar of Blank query add the code : = DateTime.LocalNow()

  • Then in the Home ribbon click on Convert to Table button. You will see the table is created.

  • Rename the Column of the table as ‘Date time’.
