Adding Average Days to Grade Visual to the Dashboard

1.Go to Visualization tab and drag and drop Line and stacked column chart visual to the canvas.

2.Go to Fields tab and add the following

    Shared Axis  : For the Week( Date Table)

                           DayOfWeek (Date Table)

  Column Series : Measures Table (.Events Graded Per Week) and Rename it to Graded Per Wk 

  Line Values : ‘Measures Table’(DaysToGrade) Rename it to Days to Grade 


3.Go to Format tab and apply the following formatting under each tab as mentioned

  •  General:  Turn Responsive on ,X Position =721, Y Position = 419, Width = 543, Height =290
  •   X Axis: Color =#605E5C, Text Size = 9, Font family = Candara,  Title = Off

  •  Y Axis : Color = #605E5C, Text size = 9, Font family = Candara,  Title = Off,  Gridlines = Off, Show Secondary= on,       Align Series =On,

  •  Data Colors : Days To Grade = #B02C84,Default color : Click on fx and  in the format style :Gradient ,what field should we base this on : Average Events Graded Per Week,How Should we format empty value=As zero,Minimum=Lowest Value,Color=#41A4FF,Maximum=Highest Value,Color =#094780
  • Data Labels : Color = #666666,Display units : Auto,Value Decimal places = Auto,orientation= Horizontal,Position =Auto,Text Size = 9,Font Family = segoe UI,Show background = on,Background color =#FFFFFF ,Transparency = 30%,customize series =on,Graded Per Week,show = on,color = #666666 Display units = Auto,Value Decimal places = Auto

  •  Shapes:  Show marker = On


    Title : On, Title Text = Average Days To Grade Event, Word wrap = On, font color = #FFFFFF, Background color = #000000, Alignment = Center, Text Size = 14, Font Family = Segoe UI Semibold
